#  Weather App

A modern and responsive Weather Application built using React that provides real-time weather updates for any city around the world.


##  Live Demo

 https://adityashrikant.netlify.app/


##  Features

*  Search weather by city name
*  Real-time temperature display
*  Fully responsive design
*  Fast and smooth user experience

---

## Tech Stack

* React.js
* JavaScript (ES6+)
* CSS
* Weather API




##  API Integration

This app uses a weather API to fetch real-time data based on user input.
